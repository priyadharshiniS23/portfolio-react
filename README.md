# 🌐 Personal Portfolio Website

## 📌 Overview

-> This is my personal portfolio website built using modern web technologies to showcase my skills, projects, and professional profile.
It includes interactive UI, project showcases, and a contact system connected with a backend and database.

------------------------------------------------------------------------------------------------------------------------------------

## 🚀 Features

- 🎯 Interactive and responsive UI
- 🧑‍💻 About Me section with profile details
- 💡 Technical Skills display (dynamic)
- 🎓 Education & Experience timeline
- 📂 Projects showcase with GitHub links
- 📄 Resume access via Google Drive
- 📬 Contact form with backend + database integration
- 🌌 Custom UI effects (mouse-based fog effect)

---------------------------------------------------------------------------------------------------------------------------------------

### Frontend:
- HTML

### Backend:
